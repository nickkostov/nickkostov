## Change Directory


# `cd` Command Tricks with Examples

### 1. Go home

```terminal-input
cd
pwd
_output_:
/home/user
```

```terminal-input
cd ~
pwd
_output_:
/home/user
```

### 2. Previous directory
```terminal-input
cd -
pwd
_output_:
/home/user
```
